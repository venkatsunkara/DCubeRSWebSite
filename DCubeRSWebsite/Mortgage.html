Detailed description

This application calculates the monthly mortgage payment based on the loan amount, term and interest rate. It can handle additional inputs like extra monthly payments and other monthly expenses. It will also show the amortization schedule and has a payment summary page. It also has pie chart that will show the total payments and bar charts that will show the principal remaining and the cumulative interest during the course of the loan term. It has several options for compounding period, payment frequency for calculations. The app has built in help along with easy to use application bar and menu options. Though the user interface text is English only the numbers, date and currency formats are displayed using the current region.
